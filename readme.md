# swole [![Build Status](https://travis-ci.org/bendrucker/swole.svg?branch=master)](https://travis-ci.org/bendrucker/swole)

> HTTP request router for Swagger/OpenAPI


## Install

```
$ npm install --save swole
```


## Usage

```js
var swole = require('swole')

swole('input')
//=> output
```
